- Versão 0.1
- Compatível com sistemas debian

Requisitos
- Debian 32 ou 64 bits
- Python 2.6 ou superior
- Python-easygui
- Python-mysqldb
- MySQL

Instalação
- Editar transportadora.conf para seus parâmetros do MySQL
- Executar o script install.sh como root

Execução
- Como usuário, executar transportadora.py
